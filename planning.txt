

- Given a map, set of actions, and corresponding sensor readings -> return PD of where you are in the world

- Data Structures
    - TransitionModel
        - dict = {action: transition_matrix}
        - getTransition(string action) -> transition_matrix
        - registerAction(string, matrix)
    - ObservationModel
        - dict = {observation: observation_model}
        - getObservation(string observation) -> observation_model
        - registerObservation(string, observation)
    - StateEstimate
        - Transition Model
        - Observation Model
        - Prior Probability Matrix
        - executeFiltering() -> PM
